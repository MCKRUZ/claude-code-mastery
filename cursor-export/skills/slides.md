# Slides

Strategic HTML presentation design with data visualization.

## When to Use

- Marketing presentations and pitch decks
- Data-driven slides with Chart.js
- Strategic slide design with layout patterns
- Copywriting-optimized presentation content

## Subcommands

| Subcommand | Description |
|------------|-------------|
| `create` | Create strategic presentation slides |

## Knowledge Base

| Topic | Description |
|-------|-------------|
| Layout Patterns | Slide layout patterns and grid systems |
| HTML Template | Base HTML template for presentations |
| Copywriting Formulas | Proven copywriting formulas for slide content |
| Slide Strategies | Contextual strategies for different slide types |

## Workflow

1. Parse the user's request for topic, audience, and slide count
2. Select appropriate layout patterns for the content type
3. Apply copywriting formulas to slide headlines and body text
4. Generate Chart.js visualizations for data-driven slides
5. Produce a single self-contained HTML file with inline CSS and JS
6. Use design tokens for consistent styling throughout

## Slide Design Principles

- **One idea per slide** -- avoid information overload
- **Visual hierarchy** -- headline, supporting data, takeaway
- **Consistent design tokens** -- colors, fonts, spacing across all slides
- **Data visualization** -- use Chart.js for bar, line, pie, and doughnut charts
- **Responsive layouts** -- slides should work on different screen sizes
- **Copywriting formulas** -- use proven structures like PAS (Problem-Agitate-Solve), AIDA (Attention-Interest-Desire-Action), and Before-After-Bridge

## Output

A single self-contained HTML file with:
- Inline CSS with design tokens
- Chart.js loaded via CDN
- Keyboard navigation (arrow keys)
- Print-friendly styles
- Responsive design
