# Research Checklist

> Systematic protocol for updating the knowledge base. Follow this checklist during each research session.

## Pre-Research

- [ ] Note current date and last update date from `references/knowledge-base.md`
- [ ] Read `references/knowledge-base.md` to understand current state
- [ ] Identify time gap since last update

## Version & Release Tracking

- [ ] What is the latest Claude Code CLI version? (Check GitHub releases)
- [ ] What changed in each version since last update? (Read changelogs)
- [ ] Any new model releases or updates? (Check Anthropic blog)
- [ ] SDK version updates? (Check npm @anthropic-ai scope)

## Feature Changes

- [ ] Any changes to CLAUDE.md behavior or hierarchy?
- [ ] New slash commands or keyboard shortcuts?
- [ ] Changes to context window sizes or compaction behavior?
- [ ] New hook events or hook types?
- [ ] Permission system changes?
- [ ] Settings schema updates?

## Agent Teams Evolution

- [ ] Still experimental or moved to stable?
- [ ] New agent team capabilities?
- [ ] File locking implemented?
- [ ] Session resumption support?
- [ ] Cost/token changes?
- [ ] New frontmatter fields for agent definitions?

## Skills & Plugins

- [ ] New official skills released?
- [ ] Skills system API changes?
- [ ] New frontmatter fields supported?
- [ ] Plugin marketplace growth?
- [ ] Notable community skills worth mentioning?
- [ ] Cross-platform skill compatibility changes?

## MCP Ecosystem

- [ ] New official MCP servers?
- [ ] Existing MCP server major updates?
- [ ] MCP protocol version changes?
- [ ] New authentication methods?
- [ ] Performance improvements?

## Community Discoveries

- [ ] New workarounds for known issues?
- [ ] Configuration patterns validated at scale?
- [ ] Anti-patterns identified?
- [ ] Cost optimization strategies?
- [ ] Enterprise deployment learnings?

## CI/CD & Automation

- [ ] GitHub Actions updates?
- [ ] New headless mode capabilities?
- [ ] Multi-session management improvements?
- [ ] Git worktree workflow updates?

## Platform-Specific (Windows Native)

- [ ] Windows-specific stability improvements in new versions?
- [ ] Git Bash detection improvements?
- [ ] Image paste (clipboard) functionality fixed?
- [ ] VS Code extension Windows compatibility updates?
- [ ] PowerShell-specific fixes or features?
- [ ] WinGet package update timing improvements?
- [ ] Windows Defender / antivirus interaction improvements?
- [ ] Windows Terminal integration updates?

## Post-Research

- [ ] Update `references/knowledge-base.md` with dated entries at top
- [ ] Update `references/changelog.md` with summary of changes
- [ ] Flag any breaking changes or urgent user-facing impacts
- [ ] Note any items that need follow-up in next research session
