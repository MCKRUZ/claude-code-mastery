# Performance

## Context Window
Avoid last 20% for large refactoring or multi-file features. Single-file edits and docs are fine near the limit.

## Research Time Limits
If a single site or source takes more than 5 minutes to fetch or return useful results, abandon that source and move on. The overall research can continue — just don't get stuck on one unresponsive or unhelpful site. One or two attempts per source max, then pivot to the next.

## Build Troubleshooting
Build fails -> use build-error-resolver agent -> analyze errors -> fix incrementally -> verify.
