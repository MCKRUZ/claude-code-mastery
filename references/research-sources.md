# Research Sources

> Where to look when updating the knowledge base. Ordered by reliability and signal quality.

## Tier 1 — Official Sources (Always Check First)

| Source | URL | What to look for |
|--------|-----|-----------------|
| Claude Code Docs | https://docs.anthropic.com/en/docs/claude-code/ | Feature announcements, config changes, new capabilities |
| Claude Code Docs Map | https://docs.anthropic.com/en/docs/claude-code/claude_code_docs_map.md | New doc pages = new features |
| Anthropic Blog | https://www.anthropic.com/news | Major releases, product direction |
| Claude Code GitHub | https://github.com/anthropics/claude-code | Release notes, issues, discussions |
| Claude Code Changelog | https://github.com/anthropics/claude-code/releases | Version-by-version changes |
| Skills Repo | https://github.com/anthropics/skills | Official skill examples and patterns |
| Agent Skills Best Practices | https://docs.claude.com/en/docs/agents-and-tools/agent-skills/best-practices | Authoring guidance |

## Research Cadence

- **Weekly:** Check GitHub releases page for new Claude Code versions
- **Bi-weekly:** Scan reddit, HN for community discoveries
- **Monthly:** Full review of all Tier 1 sources for missed changes
- **On-demand:** When user reports something unexpected or asks about a feature
